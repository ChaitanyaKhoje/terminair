## Tech Debt (non-blocking)

### Phase 4: Screens

**TD-01 (DAT-06 / SCR-01/02/04): Previous-snapshot signals not threaded to screens**
All three screens call `RegressionAnalyzer(...).analyze()` without a `previous=` argument. `grain_added`, `grain_removed`, and `upstream_schema_change` are always absent from TUI output even when `run_results_previous_path` is configured. The analyzer supports all 6 types; the threading gap is in the screen → provider interface. Affected: `model_list.py:171`, `problems.py:70`, `detail.py:151`.

**TD-02 (SCR-01): Clock does not fire on initial mount**
`_update_header()` is wired to `on_screen_resume` only. First TUI launch shows static header until user navigates away and returns. Fix: also call `self._clock_timer = self.set_interval(1.0, self._update_header)` in `on_mount()`.

### Phase 3: Config + CLI

**TD-03 (CFG cold-start UX): No --demo hint when config is absent**
Running `terminair` bare raises `ValueError: Connection 'default' not found` with no guidance toward `--demo`. Fix: catch the error in `cli.py` and append "Tip: run with --demo to start without a config file."

### Phase 1: Cleanup (docs only)

**TD-04 (CLN docs): REQUIREMENTS.md tracker stale for CLN-02 and CLN-03**
Tracker shows `[ ]` (pending); filesystem confirms both satisfied. Fix: update checkbox to `[x]` in REQUIREMENTS.md.
