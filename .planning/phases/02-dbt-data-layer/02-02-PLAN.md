var() extraction regex (LOCKED decision from CONTEXT.md):
```python
import re
_VAR_PATTERN = re.compile(r"""var\(['"](\w+)['"](?:,\s*([^)]+))?\)""")

def _extract_vars(raw_code, compiled_code):
    sql = raw_code or compiled_code or ""
    result = {}
    for match in _VAR_PATTERN.finditer(sql):
        var_name = match.group(1)
        default = match.group(2)
        result[var_name] = (default.strip().strip("'\"") if default else "REQUIRED")
    return result
```

**Class ManifestLoader:**

`__init__(self, manifest_path: Path) -> None`:
- If manifest_path does not exist, raise `FileNotFoundError(f"manifest.json not found: {manifest_path}")`
- Load JSON: `with open(manifest_path) as f: data = json.load(f)`
- Store: `self._nodes: dict[str, dict] = data.get("nodes", {})`
- Store: `self._parent_map: dict[str, list[str]] = data.get("parent_map", {})`
- Store: `self._child_map: dict[str, list[str]] = data.get("child_map", {})`
- Log: `_log.debug("ManifestLoader loaded %d nodes from %s", len(self._nodes), manifest_path)`

`get_node(self, node_id: str) -> dict | None`: return `self._nodes.get(node_id)`

`get_nodes_by_tag(self, tag: str) -> list[dict]`: return `[n for n in self._nodes.values() if tag in n.get("tags", [])]`

`get_all_tags(self) -> list[str]`: collect all tags across all nodes, return sorted unique list

`get_upstream_deps(self, node_id: str) -> list[str]`: return `self._parent_map.get(node_id, [])` — fall back to `node["depends_on"]["nodes"]` if parent_map is empty for that node

`get_downstream_deps(self, node_id: str) -> list[str]`: return `self._child_map.get(node_id, [])`

`get_full_lineage(self, node_id: str, depth: int = -1) -> dict`: recursive dict traversal building `{"node_id": str, "upstream": [lineage_dicts], "downstream": [lineage_dicts]}`. Use depth parameter: if depth == 0, return node only (no children). If depth == -1, unlimited. Guard against cycles with a `visited: set[str]` parameter. dbt DAGs are acyclic but defensive guard prevents infinite recursion if called on malformed data.

`get_grain_columns(self, node_id: str) -> list[str]`: implement the 4-step precedence exactly as in CONTEXT.md (unique_key str → unique_key list → partition_by.field → []). Node retrieved with `self._nodes.get(node_id, {})`.

`get_refs(self, node_id: str) -> list[str]`: extract from `node["refs"]` — each element is `[model_name]` (list-of-one); return flat list of names

`get_sources(self, node_id: str) -> list[str]`: extract from `node["sources"]` — each element is `[source_name, table_name]`; return `"source_name.table_name"` formatted strings

`get_dbt_vars(self, node_id: str) -> dict[str, str]`: get `node.get("raw_code")` and `node.get("compiled_code")`; apply `_VAR_PATTERN` against `raw_code or compiled_code or ""`; return dict of var_name → default_or_"REQUIRED"

`get_config(self, node_id: str) -> dict`: return `self._nodes.get(node_id, {}).get("config", {})`

`get_all_node_ids(self) -> list[str]`: return `list(self._nodes.keys())`

`build_tag_index(self) -> dict[str, list[str]]`: return `{tag: [node_id for node_id, node in self._nodes.items() if tag in node.get("tags", [])] for tag in self.get_all_tags()}`

**Class ArtifactReader:**

`__init__(self, results_path: Path, previous_path: Path | None = None) -> None`:
- If results_path does not exist, raise `FileNotFoundError(f"run_results.json not found: {results_path}")`
- Load current: `with open(results_path) as f: data = json.load(f)`
- Build index from the LIST: `self._index: dict[str, dict] = {r["unique_id"]: r for r in data.get("results", [])}`
- Log: `_log.debug("ArtifactReader loaded %d results from %s", len(self._index), results_path)`
- Handle previous file:
  ```python
  self._prev_index: dict[str, dict] | None = None
  if previous_path is not None and previous_path.exists():
      with open(previous_path) as f:
          prev_data = json.load(f)
      self._prev_index = {r["unique_id"]: r for r in prev_data.get("results", [])}
      _log.debug("ArtifactReader loaded %d previous results", len(self._prev_index))
  ```

`get_result(self, node_id: str) -> dict | None`: return `self._index.get(node_id)`

`get_previous_result(self, node_id: str) -> dict | None`:
- If `self._prev_index is None`, return `None`
- Else return `self._prev_index.get(node_id)`

`get_all_node_ids(self) -> list[str]`: return `list(self._index.keys())`

**Helper method** `get_rows_affected(self, node_id: str) -> int | None`:
- `result = self.get_result(node_id)`
- If None, return None
- `return result.get("adapter_response", {}).get("rows_affected")`

**Helper method** `get_execution_time(self, node_id: str) -> float | None`:
- `result = self.get_result(node_id)`
- If None, return None
- `return result.get("execution_time")`

**Helper method** `get_timing(self, node_id: str) -> tuple[str | None, str | None]`:
- Returns `(started_at, completed_at)` from the "execute" timing step
- `timing = result.get("timing", [])`
- `execute_step = next((t for t in timing if t["name"] == "execute"), None)`
- Return `(execute_step["started_at"] if execute_step else None, execute_step["completed_at"] if execute_step else None)`
