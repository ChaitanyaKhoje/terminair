```python
"""RegressionAnalyzer — detects row count and grain regressions across dbt model runs."""

from __future__ import annotations

from terminair.dbt.models import ModelState, RegressionSignal, Severity

_SEVERITY_ORDER = {Severity.CRITICAL: 0, Severity.WARNING: 1, Severity.INFO: 2}
```

**Class RegressionAnalyzer:**

`__init__(self, current: list[ModelState], manifest=None) -> None`:
- Store `self._current = current`
- `manifest` param is optional (for upstream_schema_change detection, pass the current ManifestLoader)

`def analyze(self, previous: list[ModelState] | None = None) -> list[RegressionSignal]`:
- `signals: list[RegressionSignal] = []`
- Build previous lookup: `prev_map = {m.node_id: m for m in previous} if previous else {}`
- For each model in self._current:
  - **row_drop / row_spike** (if model.row_delta_pct is not None):
    - delta = model.row_delta_pct
    - If delta < -30.0: append CRITICAL row_drop signal
    - Elif delta < -10.0: append WARNING row_drop signal
    - If delta > 50.0: append WARNING row_spike signal
  - **new_model_no_baseline** (if model.rows_previous is None AND model.status == "success"):
    - Append INFO new_model_no_baseline signal
  - **grain_added / grain_removed** (if model.node_id in prev_map):
    - prev_grain = prev_map[model.node_id].grain_columns
    - curr_grain = model.grain_columns
    - If len(curr_grain) > len(prev_grain): append WARNING grain_added
    - If len(curr_grain) < len(prev_grain): append CRITICAL grain_removed
  - **upstream_schema_change** (if any upstream node in prev_map and that node's grain or materialization changed):
    - For each dep_id in model.upstream_deps:
      - dep_node_name = dep_id.split(".")[-1]
      - Find in current: curr_dep = next((m for m in self._current if m.node_id == dep_id), None)
      - Find in previous: prev_dep = prev_map.get(dep_id)
      - If both exist and (curr_dep.materialization != prev_dep.materialization or curr_dep.grain_columns != prev_dep.grain_columns):
        - Append WARNING upstream_schema_change for the current model
- Sort: `signals.sort(key=lambda s: _SEVERITY_ORDER[Severity(s.severity)])`
- Return signals

`def signals_for_model(self, node_id: str) -> list[RegressionSignal]`:
- Return signals where signal.node_id == node_id
