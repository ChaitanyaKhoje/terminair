| T-03-01 | Information Disclosure | cli.py --password flag | accept | TERMINAIR_PASSWORD env var is the preferred path; --password is documented fallback; visible in ps aux only during CLI invocation, consistent with existing behavior |
| T-03-02 | Information Disclosure | config.yaml snowflake.password | mitigate | _expand_env_vars() expands ${SNOWFLAKE_PASSWORD} so plaintext never required in config file; this is the established contract and is already correct |
| T-03-03 | Tampering | --manifest / --run-results path flags | accept | click.Path(path_type=Path) normalizes the path; ManifestLoader and ArtifactReader are read-only consumers; no write surface exists |
| T-03-04 | Information Disclosure | FlashBar warning messages | accept | Warning messages contain only path strings sourced from developer's own config — no secrets exposed |
